## Project description

**Spring-JavaFX** is a Java 17 application combining **Spring Boot 3.5.0** and **JavaFX 19**.  
The project includes a Spring Boot backend (REST/web layer, data access, validation) and a desktop UI built with JavaFX.

It uses an embedded **H2** database for data storage and **Lombok** to reduce boilerplate code.  
The project is built with **Maven**.
